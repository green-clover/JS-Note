Loops in JS

JavaScript မှာ loop ပတ်လို့ရတဲ့ နည်းလမ်းလေးတွေပြောပြမယ်

for
while
do while
for in
for of
forEach
map
အဲ့တာလေးတွေသုံးပြီး loop ပတ်လို့ရတယ်

for က ဘယ်အချိန်သုံးလဲဆိုရင် အစ နဲ့ အဆုံး သေချာသိနေတာမျိုးဆိုသုံးတယ်
for မှာဆိုရင် start , end, 
step (i++ -> 1 တိုးသွားတာ, i += 5 -> 5တိုးသွားတာ)
i++ က i += 1 နဲ့တူတယ်
i += 5 က i = i + 5 နဲ့ အတူတူပဲနော်


for (let i = 0; i < 100; i++) {
  console.log("i " + i);
}
